# Projeto

Construir uma aplicação mobile de e-commerce.

## Funcionalidades

- Cadastro
- Login
- Recuperação de senha
- Catálogo
- Busca
- Favoritos
- Carrinho
- Checkout
- Pedidos
- Perfil

## Processo Obrigatório

Antes de implementar:

1. Elaborar PRD.
2. Elaborar SDD.
3. Elaborar SPECS.
4. Elaborar modelo de domínio.
5. Elaborar arquitetura.
6. Elaborar plano de implementação.

Somente após aprovação iniciar código.

## Estratégia de Implementação

Implementar em fases.

Fase 1:

- Tema global

PARAR.

- Componentes base

PARAR.

Fase 4:

PARAR.

PARAR.

PARAR.

PARAR.

## Regra

Após cada fase:

- gerar testes;
- gerar commit;
- aguardar aprovação.

Nunca continuar automaticamente.

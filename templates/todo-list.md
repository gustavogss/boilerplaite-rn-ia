# Projeto

Construir uma aplicação de lista de tarefas.

## Requisitos

O sistema deve permitir:

- Adicionar item
- Editar item
- Excluir item
- Marcar item como concluído
- Barra de progresso
- Estatísticas
- Persistência local
- Light Mode
- Dark Mode

## Workflow Obrigatório

Seguir:

PRD
↓
SDD
↓
SPECS
↓
Security Review
↓
Implementation Plan
↓
Implementação

Não gerar código antes da aprovação da arquitetura.

## Entregas Incrementais

Implementar apenas uma feature por resposta.

Exemplo:

1. Theme Provider
2. Button
3. Input
4. Todo Schema
5. Todo Storage Service
6. Todo List Screen

Parar após cada etapa.

Aguardar confirmação para continuar.

## Proibições

- Não criar múltiplas telas na mesma entrega.
- Não implementar o CRUD completo de uma só vez.
- Não alterar arquitetura definida.

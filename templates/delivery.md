# Projeto

Construir uma aplicação mobile de delivery.

## Funcionalidades

- Cadastro
- Login
- Restaurantes
- Cardápio
- Carrinho
- Checkout
- Rastreamento
- Histórico de pedidos
- Perfil

## Workflow Obrigatório

Seguir rigorosamente:

Feature Workflow
Bugfix Workflow
Refactor Workflow
Release Workflow
Deploy Workflow

## Processo

Etapa 1:
PRD

Etapa 2:
SDD

Etapa 3:
SPECS

Etapa 4:
Architecture Review

Etapa 5:
Security Review

Etapa 6:
Implementation Plan

Etapa 7:
Code Generation

## Implementação Incremental

Implementar uma única feature por vez.

Exemplo:

1. Theme Provider
2. Restaurant Card
3. Restaurant Service
4. Restaurant List Screen
5. Product Card
6. Cart Service
7. Checkout Screen

Após cada entrega:

- parar;
- gerar commit;
- listar arquivos;
- aguardar autorização.

## Critério de Aceitação

A aplicação só pode ser considerada pronta quando:

- todos os testes passarem;
- QA aprovar;
- Release Workflow concluir;
- Deploy Workflow concluir.

# AI Development Workflow

Fluxo obrigatório:

IDEIA
→ PRD
→ SDD
→ ADR
→ Feature Spec
→ Tasks
→ Implementação
→ QA
→ Security
→ DevSecOps
→ Performance
→ Review
→ Documentação
→ Entrega

Nenhuma etapa pode ser ignorada.

Se uma etapa falhar:

Retornar para a etapa anterior.

Nenhum código pode ser gerado antes da conclusão do PRD, SDD e Feature Spec.

Todos os agentes devem consultar:

- architecture.md
- coding-standards.md
- security.md
- workflow.md

antes de executar qualquer tarefa.
